# TrTr_CMR
This repository contains the official implementation for the following paper:
TrTr-CMR: Cross-Modal Reasoning Dual Transformer for Remote Sensing Image Captioning, 
which has been submitted to IEEE TGRS.


### Environment
- Python 3.9
- Pytorch 2.3.0
- CUDA 12.1

### Dataset
In this paper, all the experimental results and comparisons are conducted on UCM-Captions, Sydney-Captions and RSICD. The dataset is available at [RSICD_optimal](https://github.com/201528014227051/RSICD_optimal).

### Testing 
```
python test_3_datasets.py
```

### Training 
```
python train_swin.py
```
